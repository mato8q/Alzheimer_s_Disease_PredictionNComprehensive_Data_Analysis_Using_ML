Exploratory Data Analysis (EDA)

Target Distribution

ข้อมูลประกอบด้วยผู้ป่วยที่ไม่ได้เป็นอัลไซเมอร์ (64.6%) และผู้ป่วยที่เป็นโรค (35.4%) สะท้อนถึง Class Imbalance ระดับปานกลาง

Key Clinical Indicators

MMSE Score: กลุ่มผู้ป่วยมีคะแนนต่ำกว่าปกติอย่างมีนัยสำคัญ (p < 0.001)
Functional Assessment: ความสามารถในการทำงานลดลงอย่างชัดเจนในกลุ่มผู้ป่วย
ADL: การใช้ชีวิตประจำวัน (Activities of Daily Living) แสดงความแตกต่างที่สำคัญ
