03 · Feature Analysis

Key Predictive Features

Identified via Random Forest Gini Importance and validated through EDA statistical tests.

Gini Importance Score
Feature importance: Functional Assessment 35%, ADL 22%, MMSE 15%, Memory Complaints 8%, Behavioral Problems 5%.
01 · GINI 0.35
Functional Assessment
The primary driver for diagnosis prediction. Captures patients' ability to perform complex daily functions.
02 · GINI 0.22
ADL Index
Activities of Daily Living scores show strong correlation with disease presence and stage progression.
03 · GINI 0.15
MMSE Score
Mini-Mental State Examination remains a vital predictor — validated by significant t-test separation (p < 0.001).
04 · GINI 0.08
Memory Complaints
Patient-reported memory issues serve as a meaningful early-stage behavioral indicator.
05 · GINI 0.05
Behavioral Problems
Presence of behavioral symptoms contributes incrementally to the model's classification decision.
